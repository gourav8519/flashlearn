FLASHLEARN DASHBOARD — DROP-IN SETUP
=====================================

1. INSTALL DEPS
---------------
pnpm add framer-motion lucide-react
# (or npm / yarn / bun)


2. COPY FILES INTO YOUR NEXT.JS PROJECT
---------------------------------------
From this folder → your repo:

  components/dashboard/        → components/dashboard/
  app/dashboard/page.tsx       → app/dashboard/page.tsx
  styles/animations.css        → styles/animations.css


3. IMPORT THE KEYFRAMES
-----------------------
In app/globals.css (top of file):

  @import "../styles/animations.css";


4. PATH ALIAS (tsconfig.json)
-----------------------------
Make sure you have:

  "paths": { "@/*": ["./*"] }


5. FIRE IT UP
-------------
pnpm dev
Open http://localhost:3000/dashboard


6. WIRE XP INTO YOUR STUDY FLOW
-------------------------------
In your study screen, after a correct answer:

  import { useXP } from '@/components/dashboard/XPProvider';

  const { awardXP } = useXP();
  // on "Good" answer:
  awardXP(10, buttonRef.current);
  // on "Easy":
  awardXP(15, buttonRef.current);

Make sure <XPProvider> wraps your whole app in app/layout.tsx
(or at least both the study screen + the header containing <XPBar />).
